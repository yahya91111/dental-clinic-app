# AI V2 — Maestro Architecture

نسخة جديدة من الذكاء، مبنية على نمط المايسترو.

## الفكرة

الذكاء = مترجم لغوي + موجّه أوامر.
الخوارزميات (في `lib/algorithms/`) = العقل الفعلي.

## المعمارية: نواة + وراثة

```
core_prompt.md  (مشترك بين كل الأدوار)
        ↓
    يرث منها
        ↓
team_leader_prompt.md   (مساعد التيم ليدر)
doctor_prompt.md        (لاحقاً)
coordinator_prompt.md   (لاحقاً)
```

## البنية

```
ai_v2/
  core_prompt.md         ← النواة المشتركة
  team_leader_prompt.md  ← مساعد التيم ليدر
  catalog.md             ← قائمة الخوارزميات (يقرأها الذكاء)
  tools.ts               ← تعريفات الأدوات (لـ Anthropic)
  index.ts               ← sendMessageV2 (entry point)
  knowledge/
    schedule_rules.md    ← Lazy — تُجلب عند الحاجة
    glossary.md          ← Lazy
```

## العلاقة مع `lib/ai/` القديم

- القديم يبقى يعمل حتى نجاح V2
- لا نلمس `lib/ai/`
- نتعلّم منه ونستفيد من أنماطه
- بعد نجاح V2 → نحذف القديم

## Feature Flag

في `lib/ai.ts` متغيّر يحدّد أي إصدار يستخدم.
بداية V2: التيم ليدر فقط + خوارزمية واحدة (`schedule.build`).

## المبدأ الذهبي

**الخوارزميات تفعل العمل. الذكاء آخر حل.**

أي ميزة جديدة:
1. هل يمكن لخوارزمية أن تفعلها؟ → ابنِ خوارزمية
2. لا؟ → فكّر مرة ثانية
3. لا حقاً؟ → عندها فقط استخدم ذكاء
