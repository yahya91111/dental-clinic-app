# Algorithms

طبقة الخوارزميات — العقل الفعلي للتطبيق.

## الفكرة

كل منطق عمل يحتاج حساباً أو قراراً يعيش هنا، **بدون** اعتماد على الذكاء الاصطناعي.

الذكاء (في `lib/ai_v2/`) ينادي هذه الخوارزميات كمايسترو.

## البنية

```
algorithms/
  index.ts          ← يصدّر كل الخوارزميات
  schedule.ts       ← خوارزميات الجدول
  swap.ts           ← خوارزميات التبديل (لاحقاً)
  statistics.ts     ← الإحصائيات (لاحقاً)
```

## نمط التصدير

كل ملف يصدّر كائناً واحداً بنفس اسم المجال:

```typescript
export const schedule = {
  build: (...) => {...},
  detectConflicts: (...) => {...},
};
```

ويُستخدم هكذا:

```typescript
import { schedule } from '@/lib/algorithms';
await schedule.build({ weekStart, clinicId });
```

## القواعد

1. **بدون استدعاءات للذكاء** — كل شيء كود حتمي قابل للاختبار
2. **بدون UI** — الخوارزميات ترجع بيانات، الـ UI شيء آخر
3. **مع DB مباشرة** — تستخدم Supabase client عادي
4. **تعيد ملخصاً للذكاء** — حقل `summary` بالعربي قصير
