# D.C.M - Dental Clinic Management

A comprehensive mobile application for dental clinic management, built with React Native and Expo.

## About

D.C.M (Dental Clinic Management) is a professional mobile application designed to help dental practitioners efficiently manage their practice. The app provides tools for patient management, appointment scheduling, and treatment tracking.

## Features

- Patient Records Management
- Appointment Scheduling
- Treatment History Tracking
- Secure User Authentication
- Account Management with Delete Option

## Privacy & Account Deletion

Users can delete their account directly from the app:

1. Open the app
2. Go to "My Practice" screen
3. Tap on "Edit Profile"
4. Scroll down and tap "حذف الحساب نهائياً" (Delete Account Permanently)
5. Confirm deletion

**Note:** Account deletion is permanent and will remove all your data including patient records.
